## Reporting a Vulnerability

**Please report privately ‚Äî do NOT open a public issue!**  
We want to fix things before bad actors see them.

**Best way to report:**
- Go to the **Security** tab ‚Üí **Report a Vulnerability** (GitHub's private form ‚Äî super easy!)
- OR email us privately: vortex369x [at] proton.me (or your preferred secure email)

**What to include in your report:**
- Description of the bug/vulnerability
- Steps to reproduce (PoC code is super helpful!)
- Affected versions/files (e.g., which smart contract)
- Possible impact (e.g., "can drain funds" or "steal votes")

**What happens next?**
- We reply within **48 hours** to confirm we received it.
- We aim to update you every **7 days** (or faster if urgent).
- If accepted: we credit you (if you want) + fix it ‚Üí release patch.
- If not a real vuln: we explain politely why.

We love responsible disclosure!
