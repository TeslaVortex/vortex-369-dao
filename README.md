## 🚀 Get Started
### Python App
1. Get Python 3.12
2. Run: `pip install -r requirements.txt`
3. Start: `uvicorn app:app --reload`

### Smart Contracts
1. Get Foundry
2. Install: `forge install`
3. Test: `forge test`
4. Send live: `forge script script/Deploy.s.sol --rpc-url <rpc> --private-key <key> --broadcast`
